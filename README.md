# XYZReader

#### XYZReader is an Android application which allows users to read eBooks. The development of this project was part of the Udacity's Android Nanodegree. The final goal was improving the UI and UX by applying Material Design Guidelines.

## Getting Started

#### These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites

#### XYZReader was developed using Android Studio IDE so you must install it in your computer before proceeding:

###### https://developer.android.com/studio/

## Next Steps

#### You can directly proceed to clone the project to your local machine, enter Android Studio, import and build your project and start using XYZReader on your Android device.

## Compatibility

#### Minimum Android SDK: XYZReader requires a minimum API level of 19.
#### Compile Android SDK: XYZReader requires you to compile against API 27 or later.
